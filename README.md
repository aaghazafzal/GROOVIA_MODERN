# GROOVIA MODERN

A high-fidelity, modern music streaming platform built with Next.js, Tailwind CSS, and Framer Motion.

## Features
- **Modern UI/UX**: Glassmorphism, Premium Black & Purple Theme.
- **Responsive Design**: Dedicated Desktop Sidebar and Mobile Bottom Navigation.
- **Smart Architecture**: Modular components, Zustand state management.
- **API Integration**: Powered by JioSaavn unofficial API.

## Project Structure
- `FRONTEND`: Next.js 16 + Tailwind 4 App.
- `BACKEND`: (Planned)
- `ADMIN_PANEL`: (Planned)

## Tech Stack
- Framework: Next.js 16 (App Router)
- Styling: Tailwind CSS v4
- State: Zustand
- Icons: React Icons
- HTTP: Axios

## Getting Started
1. `cd FRONTEND`
2. `npm install`
3. `npm run dev`
